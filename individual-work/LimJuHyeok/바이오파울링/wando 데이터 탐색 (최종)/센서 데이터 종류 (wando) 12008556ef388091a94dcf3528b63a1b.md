# 센서 데이터 종류 (wando)

### 1. **용존 산소(Oxygen)**

- **oxygen_mpl**: 용존 산소 농도 (mg/L)
- **oxygen_per**: 용존 산소 포화도 (%)
- **oxygen_ppm**: 용존 산소 농도 (ppm)

### 2. **수소이온 농도(hydrogen)**

- **ph**: pH 값 (pH scale)
- **ph_meter**: pH 미터 값 (mV)

### 3. **전기 전도도(Conductivity)**

- **conductivity**: 전기 전도도 값 (µS/cm)
- **salinity**: 염도 값 (g/Kg)
- **tds**: 총 용존 고형물 값 (ppm)

### 4. **기타 필드**

- **latitude**: 위도 (위도 값)
- **longitude**: 경도 (경도 값)
- **temperature**: 온도 값 (°C)
- **redox**: 산화환원 전위 값 (mV)
